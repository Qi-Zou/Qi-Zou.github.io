OpenClaw 安装教程

OpenClaw 是一个强大的开源 AI 助手框架，可以帮你接管消息、管理日程、自动化各种任务。这篇教程将带你一步步完成安装和配置。

什么是 OpenClaw？

你可以把 OpenClaw理解为：

  • 🤖 私人 AI 秘书 — 帮你处理各种消息
  • 🔄 自动化中枢 — 连接飞书、微信、Telegram 等平台
  • 🧠 可定制智能体 — 按需求打造专属助手

系统要求

  • Node.js: 24.x（推荐）或 22.x LTS
  • 系统: macOS / Linux / Windows (WSL2)
  • 内存: 最少 2GB，推荐 4GB+

启用飞书

  1. 在飞书开放平台创建应用
  2. 获取 App ID 和 App Secret
  3. 添加权限：im:message、im:chat:readonly 等
  4. 在配置中填入相应信息

验证安装

正常的话会显示运行状态和版本信息。

常见问题

Q: 端口被占用？
A: 修改 gateway.port 为其他端口

Q: 模型连接失败？
A: 检查 API Key 是否正确，网络是否正常

Q: 飞书收不到消息？
A: 检查 App Secret 和权限配置

下一步

安装好后可以尝试：

  • 配置飞书/Telegram/微信
  • 设置定时任务
  • 对接语音合成
